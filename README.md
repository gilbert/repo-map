# Repo Map

A commit timeline visualizer for branches within a repo and its forks.

## Getting Started

```
$ git clone https://github.com/mindeavor/node-catapult my-project
$ cd my-project
$ npm install
$ export GITHUB_ID=...
$ export GITHUB_SECRET=...
$ npm start
```

Now visit [localhost:4000](http://localhost:4000/)
